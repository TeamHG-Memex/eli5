eli5.nn
=======

ELI5 has support for neural networks. It can explain predictions of image and text classifiers, using its impementation of Grad-CAM (Gradient-weighted Class Activation Mapping, `<https://arxiv.org/pdf/1610.02391.pdf>`_).
