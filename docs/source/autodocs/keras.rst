eli5.keras
==========

eli5 has Keras_ support - :func:`eli5.explain_prediction` explains
predictions of image and text classifiers by using an impementation of Grad-CAM (Gradient-weighted Class Activation Mapping, `<https://arxiv.org/abs/1610.02391>`_).
The function works with both Sequential model and functional Model.

.. _Keras: https://keras.io/
